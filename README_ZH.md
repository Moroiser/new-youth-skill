## Why This Exists

**The problem:**
- 算法决定你看到什么 → 信息茧房
- AI 生成内容自信但可能错误 → 幻觉危机
- 人们把思考外包给机器 → 丧失主体性

**The answer:**
不是告诉你"该怎么想"的技能。而是帮你"自己思考"的技能——帮助你成为一个让自己骄傲的人。

| # | Standard | What It Means | Opposite |
|:---|:---|:---|:---|
| 1 | **自主的** | 自己思考，不盲从 | 奴隶的 |
| 2 | **进步的** | 拥抱变化，持续学习 | 保守的 |
| 3 | **进取的** | 采取行动，不等待 | 退隐的 |
| 4 | **世界的** | 看到茧房之外 | 锁国的 |
| 5 | **实利的** | 结果重要，空谈无益 | 虚文的 |
| 6 | **科学的** | 事实优先，验证一切 | 想象的 |

## What It Does

| Feature | When to Use |
|:---|:---|
| **人格评估** | "我是新青年吗？" — 得到你的新青年指数 |
| **决策辅助** | "该选A还是B？" — 深入思考 |
| **内容审核** | "这个论证站得住脚吗？" — 检查逻辑 |
| **每日成长** | "今天干什么？" — 行动起来 |
| **视角转换** | "有没有其他角度看这个？" — 打破思维定式 |
| **行动翻译** | "我知道该做，但怎么做？" — 得到具体步骤 |

## The Origin

1915年，陈独秀在上海创办《La Jeunesse》（新青年）。这本杂志成为中国新文化运动的旗帜，向一代人呼唤民主（德先生）与科学（赛先生）。

这个技能将那种精神凝练成 AI 时代的工具：不是说教，而是觉醒。
